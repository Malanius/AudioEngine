# Audio
Simple 3D OpenAL audio engine with support ogg, wav format. Audio is wrote language C++ with minimal dependencies. 
This is part of engine https://bitbucket.org/deviantbytes/deviant-engine/wiki/Home

## Dependencies
 * Glm
 * OpenAL-Soft
 * Stb-vorbis
